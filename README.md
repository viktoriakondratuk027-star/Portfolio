### Проект №1 "Анализ данных для агенства недвижимости"
**Ход работы и результаты:**
- Был проведён анализ данных объявлений о продаже недвижимости в Санкт-Петербурге и Ленинградской области.
- Полученные выводы были основаны на сроках активности объявлений, месяце публикации и снятия, доле снятых объявлений, средней стоимости квадратного метра, площади квартиры и др.
- В ходе исследования были получены выводы, которые позволяют заказчику спланировать бизнес-стратегию и маркетинговые кампании:
1. Быстрее всего продаются небольшие квартиры: однокомнатные/двухкомнатные, с одним балконом и на средних этажах: 4-5.
2. Наибольшее количество объявлений публикуется в Санкт-Петербурге: стоимость кв. метра выше, чем в Ленобласти, больше половины продается менее чем за полгода.
3. Самый популярный месяц для публикации объявления - ноябрь, а для продажи - октябрь.

### Проект №3 "Исследовательский анализ данных о бронировании билетов на сервисе Яндекс Афиша"
**Ход работы и результаты:**
- Был реализован исследовательский анализ данных о бронировании билетов на сервисе Яндекс Афиша (обработано более 300 000 строк за период с 1 июня по 31 октября 2024 года, проведена классификация по типам событий, устройствам, регионам и сезонам, выполнена проверка двух статистических гипотез).
- Полученные выводы были основаны на данных о среднем чеке, количестве заказов, выручке по платформам-дистрибьюторам, среднем времени между заказами пользователей и др.
- В ходе исследования были получены выводы:
1. Концерты — самый стабильно востребованный тип событий.
2. Мобильное приложение — основной канал продаж.
- И даны рекомендации:
Расширить предложение концертов, сфокусироваться на развитии мобильной платформы и усилить предложение в регионах-лидерах (Каменевский регион и Североярская область).

### Проект №4 "A/B-тест для развлекательного приложения"
**Ход работы и результаты:**
- Был проведён A/B-тест для оценки влияния нового «бесконечного» алгоритма рекомендаций на поведение пользователей развлекательного приложения. Эксперимент проводился на двух группах: контрольной (старый алгоритм) и тестовой (новый алгоритм). Основной метрикой эффективности выбрана доля успешных сессий (например, сессий, завершившихся целевым действием).
- В ходе исследования были получены выводы, которые позволяют заказчику принять обоснованное решение о внедрении нового алгоритма:
1. В эксперименте участвовало 41 040 пользователей, равномерно распределённых по группам (по 20 520 человек в каждой). Длительность теста составила 9 дней с учётом среднего числа посещений пользователями приложения в день. Группы не пересекались по пользователям и были сопоставимы по количеству, что обеспечило чистоту эксперимента;
2. Внедрение нового алгоритма рекомендаций привело к росту доли успешных сессий на 1,1%. Этот прирост оказался статистически значимым;
3. На основе полученных данных можно утверждать, что новый «бесконечный» алгоритм рекомендаций полезен для пользователей и его внедрение повысит ключевую бизнес-метрику — долю успешных сессий.
